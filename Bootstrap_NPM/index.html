Bootstrap via NPM

Per procedere al'installazione di bootstrap tramite npm utilizzeremo il terminale:
1. npm init; Il suffisso -D imposta il salvataggio come devDependencies (dipendenze di sviluppo)

Le dipendenze di sviluppom servono per preparare l'ambiente di sviluppo per lavorare sul progetto.
Dunque procedo da terminale:

  • npm i -D dart-sass (installo dipendenza di sviluppo SASS sul progetto)
  • npm i -D postcss postcss cli(Post_CSS: tool capace di prendersi cura del CSS)
  • npm i -D autoprefixer(si occupa dei vendor prefix)
  • ndm i -D css-minify
